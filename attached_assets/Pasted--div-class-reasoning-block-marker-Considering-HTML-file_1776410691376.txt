## Что тебе нужно положить в папку `assets`

Например так:

```bash
/assets/btc.png
/assets/eth.png
/assets/sol.png
/assets/bnb.png
/assets/chain.png
```

Лучше использовать:
- `png` или `webp`
- прозрачный фон
- размер хотя бы `1000px+` по длинной стороне
- уже вырезанные объекты без фона

---

## Если хочешь, чтобы левитация была ещё ближе к референсу

Можно усилить 3 вещи:

### 1. Сделать разную амплитуду
Сейчас у каждого объекта уже своя анимация, но можно ещё сильнее разнести:

```css
.item-btc .float-wrap { animation-duration: 10s; }
.item-eth .float-wrap { animation-duration: 12s; }
.item-sol .float-wrap { animation-duration: 9s; }
```

---

### 2. Добавить задержки
Чтобы все не двигались синхронно:

```css
.item-btc .float-wrap { animation-delay: -1.2s; }
.item-eth .float-wrap { animation-delay: -3.1s; }
.item-sol .float-wrap { animation-delay: -2.4s; }
.item-bnb .float-wrap { animation-delay: -4.2s; }
.item-chain .float-wrap { animation-delay: -2.8s; }
```

---

### 3. Сделать glow под конкретный цвет объекта
Например для ETH:

```css
.item-eth .float-wrap {
  filter:
    drop-shadow(0 18px 30px rgba(0,0,0,0.45))
    drop-shadow(0 0 30px rgba(65, 120, 255, 0.28))
    drop-shadow(0 0 42px rgba(210, 70, 255, 0.18));
}
```

## Если изображения не отображаются
Проверь путь:

```html
<img src="./assets/btc.png" alt="">
```

Если файл лежит рядом с HTML, тогда так:

```html
<img src="btc.png" alt="">
```
